# goconv

npm link : [https://npmjs.com/package/goconv](https://npmjs.com/package/goconv)

A CLI tool to convert json to Golang structs and write it to a desired file of your choice

# Usage
<!-- usage -->
```sh-session
$ npm install -g goconv

USAGE
  $ goconv conv

```
